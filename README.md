# Brick Racer

A PlasticPlayBrick-themed pod racing game built entirely in a single HTML file.
Race your brick pod racer, dodge obstacles, and go for a high score!

## A Teaching Project

My kids made this all by themselves using Claude Code.

## How to Play

Open `BrickRacer.html` in any modern web browser — no server or build step needed.
